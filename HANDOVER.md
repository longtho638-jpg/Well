## 3. HƯỚNG DẪN DEPLOY

### A. Chuẩn bị Biến môi trường (.env)
Trên server (Vercel/Netlify), hãy thiết lập các biến sau:

```env
VITE_SUPABASE_URL=https://<project-id>.supabase.co
VITE_SUPABASE_ANON_KEY=<your-anon-key>
# Các biến Firebase (nếu còn dùng cho tính năng phụ, nếu không có thể bỏ)
VITE_FIREBASE_API_KEY=...
```

### B. Lệnh Deploy
**Vercel:**
```bash
vercel --prod
```

**Firebase Hosting:**
```bash
npm run build
firebase deploy --only hosting
```

## 5. SQL MIGRATION (QUAN TRỌNG)
Để kích hoạt tính năng **Realtime Wallet** và **Cây hệ thống F1-F7**, anh cần chạy đoạn mã SQL sau trong Supabase SQL Editor:

1.  Mở file: `supabase/migrations/20260113_recursive_referral.sql`
2.  Copy toàn bộ nội dung.
3.  Vào Dashboard Supabase -> SQL Editor -> New Query.
4.  Paste và bấm **Run**.

Tác dụng:
- Bật `REPLICATION` cho bảng `users` (để ví nhảy số tự động).
- Tạo hàm `get_downline_tree` (để lấy danh sách F1-F7 siêu tốc).
