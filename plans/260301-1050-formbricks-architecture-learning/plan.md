## Key Formbricks Patterns Applied

1. **JSON schema engine** → Agent flows, onboarding, training as JSON definitions
2. **Event targeting** → Contextual prompts/coaching based on distributor behavior
3. **Response pipeline** → Event-driven order processing with webhook dispatch
4. **Visual builder** → Admin UI for configuring agent workflows without code
5. **Action tracking** → Distributor behavior analytics for intelligent triggers
