- `nav.*`, `landing.*`, `marketingtools.*` → Need new files or move to existing modules
- `leaderdashboard.*`, `healthCoach.*` → Need new files

### 2. Numeric Prefix Keys
TypeScript objects can't have unquoted keys starting with numbers:
```typescript
// INVALID:
export const statsgrid = { 10_pit: "value" };

// VALID:
export const statsgrid = { "_10_pit": "value" };
// OR update code to use different key names
```

### 3. healthCheck vs healthcheck Naming Mismatch
- Code uses: `t('healthCheck.consultationTitle')`

- Fix: Either rename export or update code
