## 4. MO HINH KINH DOANH

### 4.1 Business model canvas

| Block | Mo ta |
|-------|-------|
| Value proposition | Platform TPCN + AI + commission network |
| Customer segments | Partners, consumers, B2B |
| Channels | Web, social, referral, events |
| Revenue streams | Product margin, platform fee, subscription |
| Key resources | Tech platform, partner network, AI |
| Key activities | Product curation, network growth, tech dev |
| Key partners | NCC, payment (PayOS), cloud (Supabase/Vercel) |
| Cost structure | COGS, commission, tech, marketing, team |
| Unfair advantage | AI + Network effect + Speed |

### 4.2 Unit economics

| Metric | Formula | Target |
|--------|---------|--------|
| AOV | Revenue / Orders | > 500K VND |
| CAC | Marketing spend / New customers | < 100K VND |
| LTV | AOV x Purchase frequency x Lifespan | > 3M VND |
| LTV:CAC | LTV / CAC | > 3:1 |
| Gross margin | (Revenue - COGS) / Revenue | > 40% |
| Payback period | CAC / Monthly margin per customer | < 3 thang |
