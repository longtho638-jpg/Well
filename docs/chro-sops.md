## 1. TUYEN DUNG

### 1.1 Quy trinh tuyen dung

| Buoc | Hanh dong | Timeline |
|------|-----------|----------|
| 1 | Nhan yeu cau tuyen dung tu hiring manager | Ngay 1 |
| 2 | Viet JD (Job Description) + phe duyet | Ngay 2 |
| 3 | Dang tuyen (LinkedIn, TopDev, ITviec, FB Groups) | Ngay 3 |
| 4 | Loc CV (screen 50 → shortlist 10) | Ngay 3-7 |
| 5 | Phone screen vong 1 (Culture fit, 20 phut) | Ngay 8-10 |
| 6 | Interview vong 2 (Technical/Role, 60 phut) | Ngay 11-14 |
| 7 | Interview vong 3 (CEO/Founder, 30 phut) | Ngay 15-17 |
| 8 | Reference check | Ngay 18-19 |
| 9 | Offer letter | Ngay 20 |
| 10 | Onboarding preparation | Truoc ngay bat dau |

### 1.2 Kenh tuyen dung

| Kenh | Doi tuong | Chi phi |
|------|-----------|---------|
| LinkedIn | Senior, chuyen gia | Free/Premium |
| TopDev | Dev, designer | Goi tuyen dung |
| ITviec | Tech roles | Goi tuyen dung |
| FB Groups | Tat ca | Free |
| Referral noi bo | Tat ca | Thuong 2-5M/hire |
| University | Intern, junior | Free/event |

### 1.3 Headcount plan (giai doan dau)

| Role | So luong | Uu tien | Timeline |
|------|---------|---------|----------|
| Frontend Dev | 1-2 | P1 | Q1 |
| Backend Dev | 1 | P1 | Q1 |
| Partner Manager | 1 | P0 | Ngay |
| CS Rep | 1-2 | P1 | Q1-Q2 |
| Content Creator | 1 | P2 | Q2 |
| Data Analyst | 1 | P2 | Q3 |

## 2. ONBOARDING

### 2.1 Checklist onboarding (tuan dau)

| Ngay | Hanh dong |
|------|-----------|
| Truoc ngay 1 | Setup email, Slack, access tools |
| Ngay 1 | Chao don, tour van phong/virtual, gioi thieu team |
| Ngay 1 | Cung cap: laptop, tai khoan, tai lieu |
| Ngay 2-3 | Doc CLAUDE.md, SOPs, design guidelines |
| Ngay 3-5 | Pair work voi buddy/mentor |
| Ngay 5 | 1:1 voi manager — set expectations |
| Tuan 2 | Bat dau task nho |
| Thang 1 | Review 30 ngay voi CHRO + manager |

### 2.2 Tai khoan & access

| Tool | Ai cap | Thoi gian |
|------|--------|-----------|
| Email cong ty | IT/CHRO | Truoc ngay 1 |
| Slack/Discord | IT | Truoc ngay 1 |
| GitHub repo | CTO | Ngay 1 |
| Supabase Dashboard | CTO | Ngay 1 (neu can) |
| Vercel | CTO | Ngay 1 (neu can) |
| Figma/Canva | CMO | Ngay 1 (neu design) |
| Admin Panel | COO | Khi phu hop |

## 3. QUAN LY HIEU SUAT

### 3.1 Chu ky danh gia

| Loai | Tan suat | Nguoi danh gia |
|------|----------|----------------|
| 1:1 weekly | Hang tuan | Manager |
| Performance review | Hang quy | Manager + CHRO |
| 360 feedback | Nua nam | Team + peers |
| Annual review | Hang nam | Manager + CEO |

### 3.2 Framework danh gia

| Tieu chi | Trong so | Thang diem |
|----------|---------|------------|
| Output/Results | 40% | 1-5 |
| Collaboration | 20% | 1-5 |
| Initiative | 20% | 1-5 |
| Growth/Learning | 10% | 1-5 |
| Culture fit | 10% | 1-5 |

### 3.3 Xu ly nhan vien kem

| Muc | Hanh dong | Timeline |
|-----|-----------|----------|
| Canh bao | Feedback cu the, set PIP | Ngay |
| PIP (Performance Improvement Plan) | Muc tieu ro rang, review hang tuan | 30-60 ngay |
| Quyet dinh | Cai thien → tiep tuc; Khong → cham dut | Cuoi PIP |
| Offboarding | Thu hoi access, thanh toan, exit interview | 1-2 tuan |
