# GitHub Discussions Guidelines

> Welcome to the WellNexus community discussions! This is the place to ask questions, share ideas, and connect with other developers.

---

## 📋 Discussion Categories

### 🙏 Q&A

Ask questions and get help from the community.

**Before posting:**
- ✅ Search existing discussions for similar questions
- ✅ Check the [Documentation](./docs/)
- ✅ Include error messages and steps to reproduce

**Example title:** "How to set up subscription billing for multi-org?"

---

### 💡 Ideas

Suggest new features or improvements.

**Template:**
```
## Problem Statement
What problem are you trying to solve?

## Proposed Solution
How do you think we should solve it?

## Alternatives Consider
What other approaches have you thought about?

## Additional Context
Any screenshots, mockups, or examples?
```

---

### 📢 Announcements

Official announcements from the WellNexus team.

- New version releases
- Breaking changes
- Community spotlights
- Events and meetups

---

### 🤝 Show and Tell

Share what you've built with WellNexus!

**Post includes:**
- 📸 Screenshots or demo video
- 🔗 Live link (if available)
- 💻 Code snippets (optional)
- 🎯 Challenges you overcame

---

### 📚 Tutorials

Share tutorials and guides for the community.

**Popular topics:**
- Setting up WellNexus for specific use cases
- Integrating with third-party services
- Customizing the UI/UX
- Deploying to different platforms

## 💬 Community Guidelines

### Be Respectful

- Treat everyone with respect
- No harassment or discrimination
- Constructive criticism only

### Stay On Topic

- Keep discussions focused on the original post
- Start new threads for new topics
- Use [code blocks](https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/creating-and-highlighting-code-blocks) for code

### No Spam

- No self-promotion without context
- No recruiting posts
- No affiliate links

### Report Issues

Use the **Report** button if you see:
- Spam or self-promotion
- Harassment or hate speech
- Misinformation
- Off-topic content

## 🎯 How to Get Help

### 1. Search First

Use the search bar to find existing discussions:
```
subscription billing multi-org
error 403 RLS policy
ai_copilot not working
```

### 2. Post a Good Question

**Bad:** "It doesn't work"

**Good:**
```
Title: "Subscription webhook returns 403 after PayOS callback"

Description:
I'm setting up subscription billing and after PayOS callback,
the webhook returns 403 Forbidden.

**Steps to reproduce:**
1. Create subscription intent
2. Complete PayOS payment
3. Webhook fires to /api/webhooks/payos
4. Returns 403

**Expected:** 200 OK and subscription activated

**Environment:**
- Node.js: 20.x
- WellNexus version: 3.0.0
- Deployment: Vercel

**Screenshots:**
[Attach webhook logs]
```

### 3. Follow Up

- Respond to answers with feedback
- Mark helpful answers with ✅
- Share what worked for you

## 🏆 Becoming a Community Leader

### Contributor Levels

| Level | Requirements | Perks |
|-------|--------------|-------|
| 🌱 Newcomer | First post | Welcome badge |
| 🌿 Active | 10 helpful posts | Contributor role |
| 🌳 Leader | 50+ posts, tutorials | Discord mod, swag |
| 🌟 Champion | Core contributor | GitHub org member |

### How to Level Up

1. **Answer questions** in Q&A
2. **Share tutorials** and guides
3. **Report bugs** with detailed reports
4. **Submit PRs** for docs fixes
5. **Help newcomers** get started
